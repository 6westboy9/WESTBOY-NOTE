# 数据类型

* @ 基本类型（也叫原始类型）
	1. String - 字符串
	2. Number - 数值 <font color="#c0504d">NaN特殊的数值类型，其含义就是：Not a Number</font>
	3. BigInt - 数值（<font color="#9bbb59">ES6新增</font>）
		1. BigInt只能用于表示整数，不能用于表示小数。
		2. BigInt与Number类型的值不能混合进行算术运算，必须显式转换为相同类型。
	4. Boolean - 布尔
	5. <font color="#c0504d">Null</font> - <font color="#c0504d">null是个对象类型</font>
	6. <font color="#c0504d">Undefined</font> - 只是声明了，但是没有赋值操作
	7. Symbol - 表示独一无二的值（<font color="#9bbb59">ES6新增</font>）
* @ 引用类型
	1. Object - 对象类型
		1. RegExp - 特殊对象类型
		2. Date - 特殊对象类型
	2. Array - 数组类型
	3. Function - 函数类型

## 判断方法

### 1.typeof

`typeof`操作符返回一个<font color="#c0504d">字符串</font>，表示未经计算的操作数的类型。

- 可以判断：Undefined、Number、BigInt、String、Boolean、Function
- 不能判断：Null、Object和Array

```js
var b;
console.log(typeof b); 				 	        // 'undefined' 只声明未赋值
console.log(typeof undefined); 				 	// 'undefined'
console.log(typeof 100); 				 	    // 'number'
console.log(typeof BigInt(1)); 				 	// 'bigint' 1.使用BigInt构造函数
console.log(typeof 1n); 				 	    // 'bigint' 2.使用BigInt字面量
console.log(typeof 'hello'); 				 	// 'string'
console.log(typeof true); 				 	    // 'boolean'
console.log(typeof function(){}); 				// 'function'

var c = null;
console.log(typeof c); 				 	        // 'object' null是个对象类型
console.log(typeof null);                       // 'object'
console.log(typeof {});                         // 'object'
console.log(typeof []);                         // 'object'

console.log(typeof typeof a);  	                // 'string' type返回结果本身是个字符串类型
```

```js
console.log(typeof undefined === 'undefined');  // true
```
### 2.instanceof

* 专门判断对象的具体类型
* `instanceof`运算符用于检测构造函数的`prototype`属性是否出现在某个实例对象的原型链上。

```js
var b1 = {
    b2: [1, 'abc', console.log],
    // 可以简化成 b3:()=>()=> 'hongjilin' --> 高阶函数相关知识
    b3: function () {
        return () => {
            return 'hongjilin'
        }
    }
}

// 使用instanceof进行对象判断
console.log(b1 instanceof Object, b1 instanceof Array)          // true false
console.log(b1.b2 instanceof Array, b1.b2 instanceof Object)    // true true
console.log(b1.b3 instanceof Function, b1.b3 instanceof Object) // true true


// 使用typeof进行对象中某属性的判断
console.log(typeof b1.b2, typeof null)                          // 'object' 'object'
console.log(typeof b1.b3 === 'function')                        // true
console.log(typeof b1.b2[2] === 'function')                     // true

// 调用对象与数组中某函数示例
b1.b2[2]('hello')                                               // 'hello'
console.log(b1.b3()())                                          // 'hongjilin'
```

## 引出相关问题

### 1.undefined与null的区别？

- `null`：Null类型，表示空值，代表一个空对象指针，它是一个特殊的对象值。
- `undefined`：Undefined类型，当声明了一个变量未初始化时，得到的就是undefined。

```js
var a;
var b = null;

console.log(typeof a); // 输出结果为object 类型为string
console.log(typeof b); // 输出结果为undefined
```
### 2.什么时候给变量赋值为null呢？

* 初始赋值，表明将要赋值为对象，可以用做约定俗成的占位符
* 结束前，让对象成为垃圾对象（被垃圾回收器回收）
### 3.严格区别变量类型与数据类型？

 - 数据的类型
	 - 基本类型
	 - 对象类型
 - 变量的类型（变量内存值的类型）
	 - 基本类型: 保存就是基本类型的数据
	 - 引用类型: 保存的是地址值（对象类型）
### 4.判断是否为空

 [参考资料](https://www.hangge.com/blog/cache/detail_2560.html)

1. @ 仅判断是否为`undefined`

```js
var a;
var b = null;

if (typeof a == 'undefined') {
    console.log('a 是 undefined');
} else {
    console.log('a 不是 undefined');
}

if (typeof b == 'undefined') {
    console.log('b 是 undefined');
} else {
    console.log('b 不是 undefined');
}
```

```
a 是 undefined
b 不是 undefined
```

2. @ 仅判断是否为`null`

```js
var a;
var b = null;

if (!a && typeof a !== 'undefined' && a !== 0) {
    console.log('a 是 null');
} else {
    console.log('a 不是 null');
}

if (!b && typeof b !== 'undefined' && b !== 0) {
    console.log('b 是 null');
} else {
    console.log('b 不是 null');
}
```

```
a 不是 null
b 是 null
```

3. @ 仅判断是否为`undefined`或着`null`

实际开发中我们可能不需要区分地这么细，而是只要简单地进行非空判断（如果变量为`undefined`或者为`null`都表示空）

```js
var a;  
var b = null;  
var c = 0;  
  
if (!a) {  
    console.log('a 为空');  
} else {  
    console.log('a 不为空');  
}  
  
if (!b) {  
    console.log('b 为空');  
} else {  
    console.log('b 不为空');  
}  
  
if (!c) {  
    console.log('c 为空');  
} else {  
    console.log('c 不为空');  
}
```

```
a 为空
b 为空
c 为空
```

如果变量有可能是数字0的话，使用上面的判断还不够严谨，可以使用下面方法：

```js
var a;
var b = null;
var c = 0;

if (!a && a !== 0) {
    console.log('a 为空');
} else {
    console.log('a 不为空');
}

if (!b && b !== 0) {
    console.log('b 为空');
} else {
    console.log('b 不为空');
}

if (!c && c !== 0) {
    console.log('c 为空');
} else {
    console.log('c 不为空');
}
```

```
a 为空
b 为空
c 不为空
```
