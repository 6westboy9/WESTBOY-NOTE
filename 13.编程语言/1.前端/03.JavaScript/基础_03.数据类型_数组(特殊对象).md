# 数组的定义

数组（array）是按次序排列的一组值。每个值的位置都有编号（从0开始），整个数组用方括号表示。

```js
var arr = ['a', 'b', 'c'];
```

除了在定义时赋值，数组也可以先定义后赋值。

```js
var arr = [];

arr[0] = 'a';
arr[1] = 'b';
arr[2] = 'c';
```

任何类型的数据，都可以放入数组。

```js
var arr = [
  {a: 1},
  [1, 2, 3],
  function() {return true;}
];

arr[0] // Object {a: 1}
arr[1] // [1, 2, 3]
arr[2] // function (){return true;}
```

上面数组arr的3个成员依次是对象、数组、函数。

如果数组的元素还是数组，就形成了多维数组。

```js
var a = [[1, 2], [3, 4]];
a[0][1] // 2
a[1][1] // 4
```

# 数组的本质

本质上，数组属于一种特殊的对象。typeof运算符会返回数组的类型是object。

数组的特殊性体现在，它的键名是按次序排列的一组整数（0，1，2...）。

```js
var arr = ['a', 'b', 'c'];
console.log(Object.keys(arr)); // [ '0', '1', '2' ]
```

由于数组成员的键名是固定的（默认总是0、1、2...），因此数组不用为每个元素指定键名，而对象的每个成员都必须指定键名。<font color="#c0504d">JavaScript语言规定，对象的键名一律为字符串，所以，数组的键名其实也是字符串。之所以可以用数值读取，是因为非字符串的键名会被转为字符串</font>。

```js
var arr = ['a', 'b', 'c'];

arr['0'] // 'a'
arr[0]   // 'a'
```

上面代码分别用数值和字符串作为键名，结果都能读取数组。原因是数值键名被自动转为了字符串。

注意，这点在赋值时也成立。一个值总是先转成字符串，再作为键名进行赋值。

```js
var a = [];

a[1.00] = 6;
a[1] // 6
```

上面代码中，由于1.00转成字符串是1，所以通过数字键1可以读取值。

对象有两种读取成员的方法：点结构（`object.key`）和方括号结构（`object[key]`）。但是，<font color="#c0504d">对于数值的键名，不能使用点结构</font>。

```js
var arr = [1, 2, 3];
arr.0 // SyntaxError
```

# 数组的length属性

数组的length属性，返回数组的成员数量。

```js
['a', 'b', 'c'].length // 3
```

JavaScript使用一个32位整数，保存数组的元素个数。这意味着，数组成员最多只有4294967295个（232 - 1）个，也就是说<font color="#c0504d">length属性的最大值就是4294967295</font>。

><font color="#c0504d">1.数组是一种动态的数据结构</font>

只要是数组，就一定有length属性。<font color="#c0504d">该属性是一个动态的值</font>，等于键名中的最大整数加上1。

```js
var arr = ['a', 'b'];
arr.length // 2

arr[2] = 'c';
arr.length // 3

arr[9] = 'd';
arr.length // 10

arr[1000] = 'e';
arr.length // 1001
```

上面代码表示，数组的数字键不需要连续，length属性的值总是比最大的那个整数键大1。另外，这也表明<font color="#c0504d">数组是一种动态的数据结构</font>，可以随时增减数组的成员。

><font color="#c0504d">2.length属性是可写的</font>

如果人为设置一个小于当前成员个数的值，该数组的成员数量会自动减少到length设置的值。

```js
var arr = [ 'a', 'b', 'c' ];
arr.length // 3

arr.length = 2;
arr // ["a", "b"]
```

上面代码表示，当数组的length属性设为2（即最大的整数键只能是1）那么整数键2（值为c）就已经不在数组中了，被自动删除了。

<font color="#c0504d">清空数组的一个有效方法，就是将length属性设为0</font>

```js
var arr = [ 'a', 'b', 'c' ];

arr.length = 0;
arr // []
```

如果人为设置length大于当前元素个数，则数组的成员数量会增加到这个值，新增的位置都是空位。

```js
var a = ['a'];

a.length = 3;
a[1] // undefined
```

><font color="#c0504d">3.由于数组本质上是一种对象，所以可以为数组添加属性，但是这不影响length属性的值</font>

```js
var a = [];

a['p'] = 'abc';
a.length // 0

a[2.1] = 'abc';
a.length // 0
```

上面代码将数组的键分别设为字符串和小数，结果都不影响length属性。因为，<font color="#c0504d">length属性的值就是等于最大的数字键加1</font>，而这个数组没有整数键，所以length属性保持为0。

><font color="#c0504d">4.如果数组的键名是添加超出范围的数值，该键名会自动转为字符串</font>

```js
var arr = [];
arr[-1] = 'a';
arr[Math.pow(2, 32)] = 'b';

console.log(Object.keys(arr)); // [ '-1', '4294967296' ]
console.log(arr);              // [ '-1': 'a', '4294967296': 'b' ]

arr.length // 0
arr[-1] // "a"
arr[4294967296] // "b"
```

上面代码中，我们为数组arr添加了两个不合法的数字键，结果length属性没有发生变化。这些数字键都变成了字符串键名。最后两行之所以会取到值，是因为取键值时，数字键名会默认转为字符串。


# 数组的空位

当<font color="#c0504d">数组的某个位置是空元素</font>，即两个逗号之间没有任何值，我们称该数组存在空位。

```js
var a = [1, , 1];
a.length // 3
```

上面代码表明，数组的空位不影响length属性。虽然这个位置没有值，引擎依然认为这个位置是有效的。

<span style="background:#fff88f">如果最后一个元素后面有逗号并不会产生空位</span>

```js
var a = [1, 2, 3,];

a.length // 3
a        // [1, 2, 3]
```

数组的空位是可以读取的，返回undefined。

```js
var a = [, , ,];
a[1] // undefined
```

<span style="background:#fff88f">使用delete命令删除一个数组成员，会形成空位，并且不会影响length属性。</span>

```js
var a = [1, 2, 3];
delete a[1];

a[1] // undefined
a.length // 3
```

上面代码用delete命令删除了数组的第二个元素，这个位置就形成了空位，但是对length属性没有影响。也就是说，length属性不过滤空位。所以，使用length属性进行数组遍历，一定要非常小心。

<span style="background:#fff88f">数组的某个位置是空位，与某个位置是undefined，是不一样的</span>。如果是空位，使用数组的forEach方法、for...in结构、以及Object.keys方法进行遍历，<font color="#c0504d">空位都会被跳过</font>。

```js
var a = [, , ,];

a.forEach(function (x, i) {
  console.log(i + '. ' + x);
})
// 不产生任何输出

for (var i in a) {
  console.log(i);
}
// 不产生任何输出

Object.keys(a)
// []
```

如果某个位置是undefined，遍历的时候就不会被跳过。

```js
var a = [undefined, undefined, undefined];

a.forEach(function (x, i) {
  console.log(i + '. ' + x);
});
// 0. undefined
// 1. undefined
// 2. undefined

for (var i in a) {
  console.log(i);
}
// 0
// 1
// 2

Object.keys(a)
// ['0', '1', '2']
```

><font color="#c0504d">总结</font>

* 空位就是数组没有这个元素，所以不会被遍历到，而undefined则表示数组有这个元素，值是undefined，所以遍历不会跳过。
* 获取空位返回结果是undefined。

# 数组的操作

## 键名是否存在

检查某个键名是否存在的运算符in，适用于对象，也适用于数组。

```js
var arr = [ 'a', 'b', 'c' ];
console.log(2 in arr)   // true
console.log('2' in arr) // true
console.log(4 in arr)   // false
```

上面代码表明，数组存在键名为2的键。由于键名都是字符串，所以数值2会自动转成字符串。

注意，<font color="#c0504d">如果数组的某个位置是空位，in运算符返回false</font>。

```js
var arr = [];
arr[100] = 'a';
console.log(100 in arr) // true
console.log(1 in arr)   // false
```

上面代码中，数组arr只有一个成员`arr[100]`，其他位置的键名都会返回false。

## 遍历方式

### 1.for

```js
// 声明一个数组，由于const的缘由，arr指向的数组地址不可改变，但地址内容可以改变，即数组中的值可以改变
const arr = [0, 1, 2, 3, 4, 5, 6, 7];
for (let i = 0; i < arr.length; i++) {
  if (i === 2) {
	continue;
  }
  if (i === 5) {
	break;
  }
  arr[i]++;
}
console.log(arr);
```

### 2.while

```js
var a = [1, 2, 3];

// 正序
var i = 0;
while (i < a.length) {
  console.log(a[i]);
  i++;
}

// 倒序
var l = a.length;
while (l--) {
  console.log(a[l]);
}
```

### 3.forEach

```js

colors.forEach(function (color) {
  console.log(color);
});

// red
// green
// blue
```

### forEach

```js
let arr = [
  {name: '张三', age: 18},
  {name: '李四', age: 20},
  {name: '王五', age: 18},
  {name: '赵六', age: 20},
  {name: '孙七', age: 21},
];

let obj = {};

function groupBy(arr, key) {
  arr.forEach(item => {
    item[key] = item[key] || [];
    item[key].push(item);
  });
}

groupBy()

console.log(obj);
```

### reduce

你可以使用JavaScript的Array.prototype.reduce方法来将数组分组。这是一种高级的方法，它可以将数组元素组合成一个单值。在这种情况下，你可以使用它来把数组元素放到一个对象中，其中对象的键是分组的条件，值是所有满足条件的元素组成的数组。

```js
const people = [
  {name: 'Alice', gender: 'female'},
  {name: 'Bob', gender: 'male'},
  {name: 'Charlie', gender: 'male'},
  {name: 'Diana', gender: 'female'},
];

const groups = people.reduce((groups, person) => {
  const key = person.gender;
  if (!groups[key]) {
    groups[key] = [];
  }
  groups[key].push(person);
  return groups;
}, {});

console.log(JSON.stringify(groups));
```

```json
{
    "female": [
        { "name": "Alice", "gender": "female" },
        { "name": "Diana", "gender": "female" }
    ],
	"male": [
		{ "name": "Bob", "gender": "male" },
		{ "name": "Charlie", "gender": "male" }
	]
}
```

上面的代码使用了reduce方法，它接受一个回调函数和一个初始值。回调函数的参数分别是每次遍历时的累计值（也就是groups变量）和当前元素（也就是person变量）。每次遍历时，回调函数都会返回一个新的累计值。在这个例子中，累计值是一个对象。

## 对没有属性名的数组进行分组

    }  
    acc[key].push(el);  
    return acc;  
  }, {});

# 类似数组的对象（非数组）

如果一个对象的所有键名都是正整数或零，并且有length属性，那么这个对象就很像数组，语法上称为“类似数组的对象”（array-like object）。

```js
var obj = {
  0: 'a',
  1: 'b',
  2: 'c',
  length: 3
};

obj[0] // 'a'
obj[1] // 'b'
obj.length // 3
obj.push('d') // TypeError: obj.push is not a function
```

上面代码中，对象obj就是一个类似数组的对象。但是，<font color="#c0504d">类似数组的对象并不是数组</font>，因为它们不具备数组特有的方法。对象obj没有数组的push方法，使用该方法就会报错。

## 根本特征

类似数组的对象的根本特征，就是具有length属性。只要有length属性，就可以认为这个对象类似于数组。但是有一个问题，<font color="#c0504d">这种length属性不是动态值，不会随着成员的变化而变化</font>。

```js
var obj = {
  length: 0
};
obj[3] = 'd';
obj.length // 0
```

上面代码为对象obj添加了一个数字键，但是length属性没变。这就说明了obj不是数组。
