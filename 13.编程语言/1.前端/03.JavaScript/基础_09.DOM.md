# Dom概述

DOM（Document Object Model，文档对象模型）是JS操作HTML文档的接口，使文档操作变得非常优雅、简便。

DOM最大的特点就是将HTML文档表示为<font color="#f79646">节点树</font>。

## Dom节点类型

节点的类型有七种：

1. `Document`：整个文档树的顶层节点
2. `DocumentType`：文档类型标签（比如：`<!DOCTYPE html>`）
3. `Element`：网页的各种HTML标签（比如：`<body>`、`<a>`）
4. `Attr`：网页元素的属性（比如：`class="right"`）
5. `Text`：标签之间或标签包含的文本
6. `Comment`：注释
7. `DocumentFragment`：文档的片段

```js
// 1.Document
document instanceof Document // true
document.nodeType // 9

// 2.DocumentType
document.doctype instanceof DocumentType // true
document.doctype.nodeType // 10

// 3.Element
document.body instanceof Element // true
document.body.nodeType    // 1

// <div class="menuLayout___BIz2L">...</div>
document.getElementsByClassName("menuLayout___BIz2L")[0] instanceof Element // true
document.getElementsByClassName("menuLayout___BIz2L")[0].nodeType // 1
```
