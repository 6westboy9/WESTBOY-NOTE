# 表单库：RHF

React Hook Form (RHF) 是一个轻量级、高性能的 React 表单处理库，核心设计理念是**非受控组件** + **原生验证**，相比传统的表单库（如 Formik），它能显著减少重渲染次数，提升表单性能，同时 API 简洁易用。

核心优势：

- 体积小（核心包仅～8KB）
- 性能优异（减少不必要的重渲染）
- 原生 HTML 验证集成
- 易于与 UI 库（Antd、MUI 等）集成
- 支持 TypeScript

* [React Hook Form](https://react-hook-form.com)
* [React Hook Form 中文网](https://react-hook-form.nodejs.cn)


# 表单验证库：Yup

Yup 是一个基于 JavaScript 的**声明式模式验证库**（Schema Validation），专门用于校验数据结构和值的合法性，最常见的场景就是表单验证。它的核心特点是：

- 语法简洁、可读性高（链式调用）
- 支持复杂的嵌套对象验证
- 可自定义验证规则和错误提示
- 能和 React Hook Form、Formik 等表单库无缝集成
- 完全支持 TypeScript

* [yup official document](https://www.npmjs.com/package/yup#yuplazyvalue-any--schema-lazy)
