# React 简明教程 - 第一节：Hello React 与状态管理

本节通过一个简单的计数器示例，带你快速了解 React 的核心概念。

## 项目结构

```
demo-01/
├── index.html    # HTML 入口文件
└── main.js       # React 组件逻辑
```

## 1. HTML 入口文件 (index.html)

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>React Demo 01</title>
    <!-- React 核心库（负责组件逻辑） -->
    <script crossorigin src="https://unpkg.com/react@18/umd/react.production.min.js"></script>
    <!-- ReactDOM 库（负责将组件渲染到浏览器 DOM） -->
    <script crossorigin src="https://unpkg.com/react-dom@18/umd/react-dom.production.min.js"></script>
    <!-- Babel 的独立版本（用于转译 JSX 语法，让浏览器能识别 React 的标签写法） -->
    <script type="text/javascript" src="https://unpkg.com/babel-standalone@6/babel.js"></script>
</head>
<body>
    <div id="root"></div>
    <script type="text/babel" src="./main.js"></script>
</body>
</html>
```

### 关键点说明：

| 库                 | 作用                            |
| ----------------- | ----------------------------- |
| **React**         | 核心库，负责组件逻辑和状态管理               |
| **ReactDOM**      | 负责将 React 组件渲染到浏览器 DOM        |
| **Babel**         | 将 JSX 语法转译为浏览器可执行的 JavaScript |
| `<div id="root">` | React 组件的挂载容器                 |

## 2. React 组件代码 (main.js)

```javascript
function AppContent() {
    // 参考资料：https://react.dev/reference/react/useState
    const [count, setCount] = React.useState(0);
    return (
        <div>
            <h1>Hello React!</h1>
            <p>Count: {count}</p>
            <button onClick={() => setCount(count + 1)}>+1</button>
            <button onClick={() => setCount(count - 1)}>-1</button>
            <button onClick={() => setCount(0)}>Reset</button>
        </div>
    );
}

// 通过 document.querySelector("#root") 获取页面中的挂载容器
const appEl = document.querySelector('#root');
// 用 ReactDOM.createRoot() 创建 React 的 "根节点"（React 18 的新渲染方式）
const root = ReactDOM.createRoot(appEl);
// 最后通过 root.render() 把 AppContent 组件渲染到挂载容器中
root.render(<AppContent/>);
```

## 3. 核心概念解析

### 3.1 函数组件

```javascript
function AppContent() {
    // 组件逻辑
    return (
        // JSX 返回的 UI
    );
}
```

函数组件是 React 中定义组件的主要方式，它是一个返回 JSX 的 JavaScript 函数。

### 3.2 useState Hook

```javascript
const [count, setCount] = React.useState(0);
```

| 部分 | 说明 |
|---|---|
| `count` | 当前状态值（初始值为 0） |
| `setCount` | 更新状态的函数 |
| `React.useState(0)` | 创建状态，0 是初始值 |

### 3.3 JSX 语法

```javascript
<p>Count: {count}</p>
<button onClick={() => setCount(count + 1)}>+1</button>
```

- `{count}` - 在 JSX 中嵌入 JavaScript 表达式
- `onClick` - 事件命名采用驼峰式（不是 onclick）
- `() => setCount(count + 1)` - 箭头函数作为事件处理器

### 3.4 React 18 渲染方式

```javascript
const root = ReactDOM.createRoot(appEl);
root.render(<AppContent/>);
```

React 18 引入了新的并发渲染机制，使用 `createRoot()` 替代了旧的 `render()` 方法。
