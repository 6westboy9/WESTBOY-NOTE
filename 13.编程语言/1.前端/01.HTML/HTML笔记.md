# HTML5新增标签

## fieldset

fieldset标签可以将表单内的相关元素分组。
fieldset标签会在相关表单元素周围绘制边框。

```html
<!DOCTYPE html>
<html>

<head> 
    <meta charset="utf-8"> 
    <title>菜鸟教程(runoob.com)</title> 
</head>

<body>
    <form>
        <fieldset>
            <legend>Personalia:</legend>
            Name: <input type="text"><br>
            Email: <input type="text"><br>
            Date of birth: <input type="text">
        </fieldset>
    </form>
</body>

</html>
```
