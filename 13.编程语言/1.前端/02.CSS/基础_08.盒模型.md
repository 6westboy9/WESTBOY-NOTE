# 是什么

官网文档：[盒模型](https://developer.mozilla.org/zh-CN/docs/Learn_web_development/Core/Styling_basics/Box_model)

**盒模型是CSS布局的核心基础。**

浏览器会把**每个HTML元素当作一个矩形盒子** 来渲染。  

一个盒子由四部分组成：`content`、`padding`、`border`、`margin`

```css
padding-top: 10px;    /* 上 */
padding-right: 20px;  /* 右 */
padding-bottom: 30px; /* 下 */
padding-left: 40px;   /* 左 */
```

>[!note] 便于记忆的方法：值的顺序是从上方开始顺时针设置，如果某个方向的值没有被明确指定，它会复用对方的值，注意当只设置一个值时，其它方向都复用此值

```css
padding: 10px 20px 30px 40px; // 上=10px 右=20px 下=30px 左=40px
```

```

    ┌───────────┐
40px│  content  │20px
    └───────────┘
        30px
```

```css
padding: 10px 20px 30px;      // 上=10px 右=20px 下=30px 左=20px(复用对方)
```

```

    ┌───────────┐
20px│  content  │20px

```

```css
padding: 10px 20px;           // 上=10px 右=20px 下=10px(复用对方) 左=20px(复用对方)
```

```

    ┌───────────┐

        10px

```

```

```

```
        10px
```

# box-sizing属性

CSS中的`box-sizing`属性定义了引擎应该如何计算一个元素的总宽度和总高度

```css
box-sizing: content-box|border-box|inherit:
```



```html
<!DOCTYPE html>
<html lang="zh-CN">

<head>
  <meta charset="UTF-8">
  <title>box-sizing: border-box 示例</title>
  <style>
    /* 全局初始化（推荐写法） */
    * {
      margin: 0;
      padding: 0;
      box-sizing: border-box;
      /* 全局统一使用 border-box */
    }

    .container {
      display: flex;
      gap: 20px;
      padding: 20px;
    }

    .box {
      width: 200px;
      height: 100px;
      padding: 20px;
      border: 5px solid;
      background: #eee;
    }

    /* 重置为默认的 content-box 做对比 */
    .box-content {
      box-sizing: content-box;
      border-color: #f00;
    }

    .box-border {
      border-color: #00f;
    }

    /* 辅助说明 */
    .desc {
      margin-top: 10px;
      font-size: 14px;
      color: #666;
    }
  </style>
</head>

<body>
  <div class="container">
    <div>
      <div class="box box-content">content-box（默认）</div>
      <p class="desc">总宽：250px（200+20*2+5*2）</p>
    </div>
    <div>
      <div class="box box-border">border-box</div>
      <p class="desc">总宽：200px（包含padding/border）</p>
    </div>
  </div>
</body>

</html>
```
