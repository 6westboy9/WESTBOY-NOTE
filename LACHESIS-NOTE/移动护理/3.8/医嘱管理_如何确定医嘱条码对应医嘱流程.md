# 医嘱流程

| 流程类型                    | 流程类型名称     | 医嘱流程判断条件                                                                                                 | 流程节点                                                                                                            |
| ----------------------- | ---------- | -------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| infusion                | 自配输液       | 存在与windranger_emr.pat_inhos_order表，usageCode=INFUSION 且 isDispensed=0                                    | medicine_prepare -> medicine_check -> dispensing -> dispensing_check -> order_execute -> infusion_inspect       |
| infusion_dispense       | 静配输液       | 存在与windranger_emr.pat_inhos_order表，usageCode=INFUSION 且 isDispensed=1                                    | medicine_receive -> order_execute -> infusion_inspect                                                           |
| infusion_other          | 泵入类输液      | 存在与windranger_emr.pat_inhos_order表，usageCode=INFUSION_OTHER                                              | medicine_prepare -> medicine_check -> dispensing -> dispensing_check -> order_execute -> infusion_inspect       |
| inject                  | 针剂注射       | 存在与windranger_emr.pat_inhos_order表，usageCode 属于IN_INJECT / INJECT / SUB_INJECT / PUMP_INJECT / IV_INJECT | medicine_prepare -> medicine_check -> dispensing -> dispensing_check -> order_execute                           |
| oral_package            | 口服用药(包药机)  | 存在与windranger_emr.pat_inhos_order表，usageCode=ORAL 且 isDispensed=2                                        | medicine_receive -> order_execute                                                                               |
| oral                    | 口服用药(非包药机) | 存在与windranger_emr.pat_inhos_order表，usageCode=ORAL 且 isDispensed!=2                                       | medicine_prepare -> medicine_check -> order_execute                                                             |
| atomization             | 雾化         | 存在与windranger_emr.pat_inhos_order表，usageCode=ATOMIZING                                                   | medicine_prepare -> medicine_check -> dispensing -> dispensing_check -> order_execute                           |
| blood                   | 输血         | 2种来源：1.存在与windranger_emr.pat_inhos_order表，usageCode=BLOOD；2.存在与windranger_emr.pat_emr_blood_transfusion表 | blood_prepare -> blood_receive -> blood_check -> blood_execute -> blood_inspect -> blood_recover                |
| check                   | 检验         | 2种来源：2.存在与windranger_emr.pat_inhos_order表，usageCode=LAB/UC；2.存在与windranger_emr.pat_lab_master表           | check_collect_start -> check_collect_end -> check_send                                                          |
| skinTest                | 皮试         | 存在与windranger_emr.pat_inhos_order表，usageCode=SKINTEST                                                    | skinTest_execute -> skinTest_input                                                                              |
| skinTestMedicinePrepare | 皮试(备药)     | 存在与windranger_emr.pat_inhos_order表，usageCode=SKINTEST_MEDICINE_PREPARE                                   | medicine_prepare  -> medicine_check ->  dispensing ->  dispensing_check ->  skinTest_execute ->  skinTest_input |
| nursing_care            | 护理治疗       | 条件1.存在与windranger_emr.pat_inhos_order表，医嘱类别是护理类别 + 条件2.usageCode不是NURSING_CARE_CHECK/NURSING_CARE_PATROL | nursing_execute                                                                                                 |
| nursing_care_check      | 护理治疗(审核)   | 条件1.存在与windranger_emr.pat_inhos_order表，医嘱类别是护理类别 + 条件2.usageCode=NURSING_CARE_CHECK                      | nursing_execute_check -> nursing_execute                                                                        |
| nursing_care_patrol     | 护理治疗（巡视）   | 条件1.存在与windranger_emr.pat_inhos_order表，医嘱类别是护理类别 + 条件2.usageCode=NURSING_CARE_PATROL                     | nursing_execute -> nursing_patrol -> nursing_finish                                                             |
| 自定义 processType         | 自定义流程      | 从查MongoDB中mnis库orderProcessTpl集合查询，按照{usageCode,current=true}条件查询，取其中的processType                        | 对应processType的流程节点                                                                                              |
说明：usageCode意思为用法组，那么用法组到底是如何根据条码确定呢？
