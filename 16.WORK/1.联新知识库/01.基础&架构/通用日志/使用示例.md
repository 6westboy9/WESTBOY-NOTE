
配置

```yaml
lachesislog:
  enable: true
  log-enable: false
  monitor-enable: false
  listener-name: default
```

使用

```java
@LachesisEntryLog(opName = "血袋回收",
	operator = "{@orderUtils.getUserNameByCode(@valueBeanStamp.getUserCodeInConext())}",
	content = "{T(cn.hutool.json.JSONUtil).toJsonStr(#p0)}",
	logId = "{T(com.yomahub.tlog.context.TLogContext).getTraceId()}")
```
