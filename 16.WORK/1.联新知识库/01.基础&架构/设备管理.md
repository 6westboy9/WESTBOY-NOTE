```java
class BasicInfo {
	private String id;                            // 主键

	private String deviceIp;                      // 设备IP
	private String uniqueSn;                      // 设备管理唯一标识

    private Date updateTime;  
    private String updatePerson;  
    private String createPerson;  
    private String fromUser;

class BasicInfoContent {
    // basicInfo.directional  ===> type:Integer | defaultValue:0 | desc:是否是增援设备 0-否 1-是
    private JSONObject basicInfo;
}
```

# 设备管理

## PC端手动添加设备


>好多内容需要与前端确认，这玩意就是JSONObject，鬼知道有哪些参数？

<font color="#c0504d">DEVICE_TYPE_DARMA类型设备</font>

```js
// basicInfo.content.basicInfo详情（存储结构为JSONObject）
{
   "mac": "$mac",                -- 设备MAC地址，$mac表示添加时请求参数传递
   "sn": "$mac"                  -- 
   "deviceIdentityCode": "$mac"  -- 
   "eqpName": "DM$mac"           -- 
   "eqpStatus": 10001,           -- 设备状态
}
```

<font color="#c0504d">非DEVICE_TYPE_DARMA类型设备</font>

```js
// basicInfo.content.basicInfo详情（存储结构为JSONObject）
{
   "defaultUrls": "",  -- 
   "serverIP": "",     -- 
   "eqpStatus": 10001, -- 设备状态
}
```
