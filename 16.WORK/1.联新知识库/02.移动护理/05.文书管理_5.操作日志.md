# 概述

| 模块                                | 描述   | 含义                                                                                  |
| --------------------------------- | ---- | ----------------------------------------------------------------------------------- |
| <font color="#c0504d">护理记录</font> | 创建文书 | 新增表头                                                                                |
| <font color="#c0504d">护理记录</font> | 修改文书 | 对单个表头的编辑操作包括：<font color="#c0504d">1.新增单/行记录；2.修改单/行记录；3.新增+修改单/行记录；4.删除整行记录</font> |
| <font color="#c0504d">护理记录</font> | 删除文书 | 对单个表头的删除操作                                                                          |
| <font color="#f79646">护理评估</font> | 创建文书 | 新建评估记录<font color="#c0504d"> -- 在文书编辑页面显示为表格的单行记录</font>                            |
| <font color="#f79646">护理评估</font> | 修改文书 | 对单个记录的编辑操作                                                                          |
| <font color="#f79646">护理评估</font> | 删除文书 | 对单个记录的删除操作                                                                          |
| <font color="#9bbb59">普通文书</font> | 创建文书 | 新建一份文书                                                                              |
| <font color="#9bbb59">普通文书</font> | 修改文书 | 对单份文书的编辑操作                                                                          |
| <font color="#9bbb59">普通文书</font> | 删除文书 | 对单份文书的删除操作                                                                          |
