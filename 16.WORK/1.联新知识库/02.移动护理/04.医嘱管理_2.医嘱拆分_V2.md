## 频次维护

频次维护属性

* 频次编号（必填）：
* 拆分类型（必填）：见下表格
* 拆分配置（必填）：<font color="#ff0000">重要</font>
* 频次名称（可选）：可不填写，一般用于配置说明

| 拆分类型   | 拆分类型编码               |
| ------ | -------------------- |
| 立即执行   | `ExecuteNow`         |
| 间隔天数   | `IntervalDay`        |
| 间隔天数逢单 | `IntervalDaySingle`  |
| 间隔天数逢双 | `IntervalDayDouble`  |
| 间隔时间   | `IntervalTime`       |
| 间隔时间逢单 | `IntervalTimeSingle` |
| 间隔时间逢双 | `IntervalTimeDouble` |
| 一周多次   | `WeekManyTimes`      |
| 一天多次   | `DayManyTimes`       |
