# 1.边界布局

```java
public class BorderLayoutExample extends JFrame {  
    JButton btn1 = new JButton("东");  
    JButton btn2 = new JButton("南");  
    JButton btn3 = new JButton("西");  
    JButton btn4 = new JButton("北");  
    JButton btn5 = new JButton("中");  
  
    BorderLayoutExample() {  
        init();  
        this.setTitle("边界布局");  
        this.setResizable(true);  
        this.setSize(200, 200);  
        this.setLocationRelativeTo(null);  
        this.setDefaultCloseOperation(EXIT_ON_CLOSE);  
        this.setVisible(true);  
    }  
  
    void init() {  
        this.setLayout(new BorderLayout(10, 5)); // 默认为0，0；水平间距10，垂直间距5  
        this.add(btn1, BorderLayout.EAST);  
        this.add(btn2, BorderLayout.SOUTH);  
        this.add(btn3, BorderLayout.WEST);  
        this.add(btn4, BorderLayout.NORTH);  
        this.add(btn5, BorderLayout.CENTER);  
    }  
  
    public static void main(String[] args) {  
        new FlowLayoutExample();  
    }   
}
```

`new GridBagLayout()`本身不会默认指定具体的行数或列数。它是一个灵活的布局管理器，根据添加的组件及其约束（`GridBagConstraints`）动态计算组件的位置和大小。

简要说明：
* GridBagLayout没有固定的“默认行数”或“列数”。
* 行和列的数量由你添加的组件以及设置的gridx、gridy、gridwidth和gridheight等约束参数决定。
* 如果未显式指定某些约束属性，GridBagLayout会按照默认规则进行布局，例如按顺序从左到右、从上到下排列。
* 如需控制行列，请合理使用GridBagConstraints来配置每个组件的布局行为。

`GridBagConstraints`中用于控制组件在`GridBagLayout`布局中位置和大小的关键参数：

| 参数名        | 含义说明                                                             |
| ---------- | ---------------------------------------------------------------- |
| gridx      | 指定组件放置的起始列（从左到右），默认值为`0`。使用`RELATIVE`表示由布局管理器自动决定下一列的位置。         |
| gridy      | 指定组件放置的起始行（从上到下），默认值为`0`。使用`RELATIVE`表示由布局管理器自动决定下一行的位置。         |
| gridwidth  | 指定组件横向跨越的列数，默认值为`1`。可以设置为具体的整数值，也可以使用`REMAINDER`表示该组件是该行的最后一个组件。 |
| gridheight | 指定组件纵向跨越的行数，默认值为`1`。可以设置为具体的整数值，也可以使用`REMAINDER`表示该组件是该列的最后一个组件。 |

>示例代码片段
 
```java
GridBagConstraints gbc = new GridBagConstraints();
gbc.gridx = 0;      // 起始列是第 0 列
gbc.gridy = 0;      // 起始行是第 0 行
gbc.gridwidth = 2;  // 横向跨越 2 列
gbc.gridheight = 1; // 纵向跨越 1 行
```
通过合理配置这些参数，你可以精确控制组件在网格中的布局方式。

# 5.空布局

空布局就是不用布局管理器，自己设置组件的顶点坐标，设置组件的高度和宽度。

![[Pasted image 20250607232857.png|L|400]]

```java
public class EmptyExample extends JFrame {  
  
    public EmptyExample() {  
        init();  
    }  
  
    public static void main(String[] args) {  
        new EmptyExample();  
    }  
  
    public void init() {  
        JButton button1 = new JButton("button1");  
        JButton button2 = new JButton("button2");  
  
        // 设置组件的位置  
        // 方法一：  
        button1.setLocation(100, 100);  
        button1.setSize(100, 100);  
        // 方法二：  
        button2.setBounds(200, 200, 100, 100);  
        Container container = this.getContentPane();  
        container.add(button1);  
        container.add(button2);  
        // 设置空布局管理器--即不设置布局管理器  
        this.setLayout(null);  
        this.setTitle("空布局学习");  
        this.setSize(500, 500);  
        this.setLocationRelativeTo(null);  
        this.setVisible(true);  
        // 设置JFrame窗体关闭时 程序结束  
        this.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);  
    }  
}
```
