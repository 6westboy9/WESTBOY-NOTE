# box-sizing属性

https://www.itheima.com/news/20210924/115317.html

```html
<!DOCTYPE html>
<html lang="zh-CN">

<head>
  <meta charset="UTF-8">
  <title>box-sizing: border-box 示例</title>
  <style>
    /* 全局初始化（推荐写法） */
    * {
      margin: 0;
      padding: 0;
      box-sizing: border-box;
      /* 全局统一使用 border-box */
    }

    .container {
      display: flex;
      gap: 20px;
      padding: 20px;
    }

    .box {
      width: 200px;
      height: 100px;
      padding: 20px;
      border: 5px solid;
      background: #eee;
    }

    /* 重置为默认的 content-box 做对比 */
    .box-content {
      box-sizing: content-box;
      border-color: #f00;
    }

    .box-border {
      border-color: #00f;
    }

    /* 辅助说明 */
    .desc {
      margin-top: 10px;
      font-size: 14px;
      color: #666;
    }
  </style>
</head>

<body>
  <div class="container">
    <div>
      <div class="box box-content">content-box（默认）</div>
      <p class="desc">总宽：250px（200+20*2+5*2）</p>
    </div>
    <div>
      <div class="box box-border">border-box</div>
      <p class="desc">总宽：200px（包含padding/border）</p>
    </div>
  </div>
</body>

</html>
```
