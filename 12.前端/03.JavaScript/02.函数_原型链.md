需着重理解以下几点：

* 构造函数只不过是通过`new`调用的函数而已
* **函数是特殊的对象**，虽然使用`typeof`返回结果是`funcation`而非`object`，构造函数也是对象
* 任何函数都有`prototype`属性，且`prototype`属性指向的对象称为<font color="#f79646">原型对象</font>，每个原型对象里面都有个`constructor`属性
* 对于普通函数来说的`prototype`属性没有任何用处，而构造函数的`prototype`属性非常有用

```js
function Sum(a, b) {
	return a + b;
}

console.log(Sum.prototype);                      // {}
console.log(typeof Sum.prototype);               // object
console.log(Sum.prototype.constructor);          // [Function: sum]
console.log(typeof Sum.prototype.constructor);   // function
console.log(Sum.prototype.constructor === Sum);  // true
```
