NodeList实例是一个类似数组的对象，它的成员是节点对象。通过以下方法可以得到NodeList实例。

# 获取NodeList实例

## 第一种

```js
document.body.childNodes
document.body.childNodes instanceof NodeList // true
document.body.childNodes[0] instanceof Element // true
```

## 第二种

```js
document.querySelectorAll('#root')
document.querySelectorAll('#root') instanceof NodeList // true
document.querySelectorAll('#root')[0] instanceof Element // true
```

## 第三种

# NodeList转数组

NodeList实例很像数组，可以使用length属性和forEach方法。但是，它不是数组，不能使用pop或push之类数组特有的方法。

```js
var children = document.body.childNodes;
console.log(Array.isArray(children)); // false

console.log(children.length);
children.forEach(console.log);
```

如果NodeList实例要使用数组方法，可以将其转为真正的数组。

```js
var children = document.body.childNodes;
var nodeArr = Array.prototype.slice.call(children);

// 除了使用forEach方法遍历NodeList实例，还可以使用for循环
for (var i = 0; i < children.length; i++) {
  var item = children[i];
}
```

# 动态与静态

NodeList实例可能是动态集合，也可能是静态集合。所谓动态集合就是一个活的集合，DOM删除或新增一个相关节点，都会立刻反映在NodeList实例。目前，<font color="#f79646">只有第一种方式返回的是一个动态集合</font>，其他的NodeList都是静态集合。

## 动态验证

```js
var children = document.body.childNodes;
console.log(children.length); // 18
document.body.appendChild(document.createElement('p'));
console.log(children.length); // 19
```

## 静态验证

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
<div>
    <ul id="list">
        <li>牛奶</li>
        <li>咖啡</li>
        <li>可乐</li>
    </ul>
</div>
<script>
  const liElements = document.querySelectorAll("li");
  console.log(liElements.length); // 3

  // 添加一个新的元素
  const newLi = document.createElement("li");
  newLi.textContent = "新项目";
  document.getElementById("list").appendChild(newLi);

  console.log(liElements.length); // 3
</script>
</body>
</html>
```
