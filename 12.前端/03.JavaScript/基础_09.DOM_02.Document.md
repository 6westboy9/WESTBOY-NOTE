# Document节点概述

document节点对象代表整个文档，每张网页都有自己的document对象。window.document属性就指向这个对象。只要浏览器开始载入HTML文档，该对象就存在了，可以直接使用。

document对象获取方式：

1. 网页：直接使用document或者window.document
2. iframe：
3. Ajax：
4. 内部节点：内部节点的ownerDocument属性

>4.内部节点方式

```html
<!DOCTYPE html>
<html lang="zh">
<head>
    <meta charset="UTF-8">
    <title>获取document对象</title>
</head>
<body>
<div id="myElement">这是一个元素</div>
<script>
  let element = document.getElementById("myElement");
  let documentObject = element.ownerDocument;

  console.log(documentObject === document); // true
</script>
</body>
</html>
```

# Document节点方法


* `document.open`：清除当前文档所有内容，使文档处于可写状态，供`document.write`方法写入内容
* `document.write`：向文档写入内容
* `document.close`：用来关闭`document.open`打开的文档


```js
document.open(); // 浏览器显示空白页
document.write("hello1"); // hello1
document.write("hello2"); // hello1hello2
document.close(); 
document.open(); // 浏览器显示空白页
document.write("hello1"); // hello1
```
